# Login video assets

Place the login background video here using one of these names:

- `luxury-hotel-login.webm` (preferred, smaller)
- `luxury-hotel-login.mp4` (fallback)

Recommended export settings:
- 8-12 seconds loop
- 1920x1080 or 1280x720
- muted/no audio track
- WebM VP9 or MP4 H.264
- keep under 5-8 MB for local demo performance
